## Push Portfolio Updates to GitHub Pages

// turbo-all

### Pre-flight: Make sure you're in the right directory
All commands run from `D:\Antigravity Projects\Portfolio Website`

### Steps

1. Check what files have changed:
```
git status
```

2. Stage all changes:
```
git add -A
```

3. Commit with a descriptive message:
```
git commit -m "your commit message here"
```

4. Push to GitHub (site updates automatically in ~1 min):
```
git push origin master
```

5. Verify the live site at: https://manoharsattaru.github.io/portfolio/
